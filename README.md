# DroneSim (PaperPiper)

A Java-based drone simulation platform for hopefully realistic quadcopter physics and swarm coordination.



## Building

```bash
# Clone the repository
git clone https://github.com/Paper-2/DroneSim.git
cd DroneSim

# Build the project
./gradlew build
```

## Running

```bash
./gradlew run
```
